## Sadaa Vijayee – Project 1

### Healthcare Data – Disease Risk Prediction

**Type:** Job Simulation | Predictive Analytics | Healthcare ML

### Problem Statement

Hospitals need early identification of patients at risk of chronic diseases. This project focused on **predicting disease risk** and providing **interactive dashboards** for healthcare decision-makers.

### Tools & Technologies

- Python (pandas, scikit-learn)
- Machine Learning (Logistic Regression, Decision Tree)
- SQL
- Power BI

### Key Tasks

- Data cleaning and feature engineering
- Exploratory data analysis
- Predictive modeling
- Patient risk segmentation using SQL
- Dashboard creation for insights

### Deliverables

- Python notebooks
- SQL queries
- Power BI dashboard
- Final strategy report
