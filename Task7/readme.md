# Visualizing Sales Trends and Customer Behavior

## **Overview**

This project analyzes the **Superstore Dataset** to understand sales performance, customer behavior, product categories, and regional trends using **Power BI**.
It demonstrates skills in **data cleaning**, **data modeling**, **DAX**, and **interactive visualization**.

## **Steps Performed**

### **Data Cleaning (Power Query)**

- Fixed data types (Order Date, Ship Date, Sales, Postal Code, IDs)
- Resolved date parsing errors using **Locale**
- Ensured consistent and clean fields

------

### **Dashboard Pages**

#### **Page 1 – Sales Overview**

- Total Sales, Orders, Customers (KPIs)
- Monthly Sales Trend
- Sales by Category & Sub-Category
- Sales by Region / State
- Slicers (Date, Region, Segment, Category)

#### **Page 2 – Customer & Product Insights**

- Sales by Segment
- Sales by Ship Mode
- Sales by State
- Best-Selling Products
- Slicers for filtering

------

## **Insights**

- Technology & Office Supplies drive the highest sales
- Consumer segment contributes the most revenue
- Western region performs strongest
- Ship Mode preference: Standard Class most used
- Seasonal trends visible in monthly sales

------

## **Tools Used**

- **Power BI Desktop**
- Power Query (for cleaning)
- DAX (for measures)
- Data Modeling

------

## **Outcome**

A fully interactive Power BI dashboard enabling:

- Sales trend analysis
- Customer behavior understanding
- Product performance review
- Regional comparison
