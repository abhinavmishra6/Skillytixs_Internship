### **Overview**

This project contains an interactive **Sales & Profit Analysis Dashboard** built using Power BI.
The dataset used is the **Superstore Sales Dataset** from Kaggle.

The dashboard helps analyze:

- Overall Sales & Profit
- Profit Margins
- Year-over-Year (YoY) performance
- Category & Sub-category contribution
- Regional & State-wise performance
- Product-level insights

------

##  **What This Report Includes**

This dashboard demonstrates:

- Data Cleaning & Transformation using Power Query
- DAX Measures (Total Sales, Total Profit, Profit Margin %, YoY Growth)
- Interactive visuals (Line charts, Bar charts, Tree map)
- Slicers for Region, Category, and Date
- Tooltips, hover effects, and consistent theming
- Final professional dashboard layout
